# Cursor Setup Assignment

Tools Installed

* Cursor IDE
* Claude Desktop (Windows)
* Codex Extension
* GitHub Desktop
* GitHub

Steps Completed

1. Installed Cursor from https://cursor.com/ on desktop.
2. Signed up for Cursor using my Google account.
3. Connected my GitHub account with Cursor.
4. Opened the editor window in Cursor.
5. Accessed the Extensions tab from the left sidebar.
6. Installed the Codex extension (OpenAI's coding agent).
7. Changed the Claude Code access shortcut to **"Claude Code: Open"**.
8. Opened GitHub in the browser and created a new account using my Google account.
9. Created a new public repository with a README file.
10. Opened Cursor and used the **Clone Git Repository** option to work with the repository.



Issues Faced and How I Solved Them

### Issue 1: Could not find the Extensions option in Cursor

Initially, I was not able to see the Extensions option in Cursor, which prevented me from installing the required extensions.

Solution:
I searched online and found that I first needed to open the editor window. After opening it, the Extensions option appeared on the left sidebar, and I was able to install extensions normally.

---

Issue 2: Unable to install Claude Code

While trying to install the Claude Code extension through its website, I received the following error:

> Cannot read properties of undefined (reading 'activeEditor')

Solution:
I searched for different solutions online. Since the installation through the website was not working, I tried installing the Claude Desktop application for Windows and also looked into manually installing the extension using the VSIX method.



Issue 3: Different interface from online tutorials

Most online tutorials showed a different Cursor interface than the one I had, making it difficult to follow the same steps.

**Solution:**
I explored the available options in my version of Cursor and searched online for alternative methods that matched the latest interface.



Issue 4: Problems connecting the GitHub repository

After completing most of the setup, I faced issues connecting my GitHub repository.

Solution:
I searched for manual methods and installed GitHub Desktop to connect the repository. Although it took some extra troubleshooting, I was able to continue with the setup process.
